Podemos também criar listas mistas, que são listas dentro de outras listas, como no exemplo abaixo:

  1. item 1
  2. item 2
    1. item 2.1
    2. item 2.2
    3. item 2.3
  3. item 3
  4. item 4
  5. item 5

Também podemos misturar listas ordenadas com listas não ordenadas, exemplo:

  1. item 1
  2. item 2
    • item 2.1
    • item 2.2
    • item 2.3
  3. item 3
  4. item 4
  5. item 5
